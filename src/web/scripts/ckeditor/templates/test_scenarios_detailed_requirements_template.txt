<h2><strong style="color: rgb(192, 0, 0); font-size: 13px; line-height: 1.6em;">Project Overview</strong></h2>

<p><span style="color:#002060;"><em style="line-height: 1.6em;">[Provide an overall description of the project. Include any tips for the success of the project.]</em></span></p>

<p><strong style="line-height: 1.6em;"><em>Example:</em></strong></p>

<p><span style="line-height: 1.6em;">Styx is a company that sells software applications to customer. The client is looking to create a new Windows-based application for use by the company&#39;s Technical Support personnel to help troubleshoot and identify issues present within the customers&#39; environments that prevent the original system from functioning. The application will need to have the ability to perform the following basic operations:</span></p>

<ul>
	<li>&nbsp;Perform network connectivity tests between machines</li>
	<li>Provide a profile of the network bandwidth and latency</li>
	<li>Perform a stress test on the customer&#39;s network to identify the point when collision and re-transmissions begin increasing</li>
	<li>Report on and compare against a predefined set of rules the DCOM configuration settings</li>
	<li>Ability to collect network metrics over a length of time</li>
	<li>Provide an easy, intuitive report of the metrics in an existing format (PDF), with an option to e-mail report to Tech Support</li>
	<li>The application must have an easy-to-use, intuitive user interface and logic to be able to make a determination if the collected data does not match the company&#39;s prerequisites for proper operation</li>
	<li>The application will need to have basic help descriptions to allow new users to learn how to use these tools</li>
</ul>

<p>&nbsp;&nbsp;&nbsp; (Note that additional functionality may be exposed during the requirements gathering process.)</p>

<p>This competition will define the test scenarios for the Network Analyzer Tool.</p>
